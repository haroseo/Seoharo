## 📬 Discord Webhook 연동 (문의하기)

### 설정
- 환경변수명: VITE_DISCORD_WEBHOOK_URL
- 로컬: .env.local 파일
- 배포: GitHub Secrets에 등록

### 작동 방식 (ContactPage.tsx)
```typescript
const webhookUrl = import.meta.env.VITE_DISCORD_WEBHOOK_URL;
await fetch(webhookUrl, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    embeds: [{
      title: "새 문의: " + formState.name,
      description: formState.message,
      fields: [{ name: '이메일', value: formState.email, inline: true }],
      color: 0xffffff,
      timestamp: new Date().toISOString()
    }]
  })
});
```

1. "시발점" 단어 사용 금지 — Limited™ 설명에서 이상해서 삭제 요청
2. "론칭" 표현 금지 → 반드시 "제작" 또는 "구현"으로

5. 영어로만 또는 한국어로만 극단적으로 쓰지 말 것

1. 숫자에 항상 "+" 붙이기 — 1,200+ / 30+ / 3+ 등 크게 보이게
2. 경력 수는 "30+"로 표시 — 30곳 이상 실무 경험 보유
3. 작업 완료 후 항상 3가지 질문:
   - 더 만들 것
   - 수정할 것
   - 예시/제안 요청
4. KO/EN 이중 언어 지원 유지
5. 이미지는 사용자가 준 실제 자산만 사용
