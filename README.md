# SEOHARO Portfolio ✨

안녕하세요! **SEOHARO (서하루)**입니다.  
시각적인 아름다움(Brand Design)과 기술적 구현(Programming)을 결합하여, 단순한 아이디어를 가치 있는 브랜드 경험으로 만들어가는 디자이너 겸 CEO입니다.

이 레포지토리는 저의 작업물과 디자인 철학, 그리고 운영해 온 커뮤니티의 발자취를 담아낸 개인 포트폴리오 웹사이트 소스 코드입니다.

---

## 🚀 About This Project

이 포트폴리오 웹사이트는 저의 디자인 아이덴티티를 가장 잘 보여줄 수 있도록 기획부터 디자인, 개발까지 직접 진행했습니다. 
깔끔한 UI와 프리미엄한 타이포그래피(Pretendard & Outfit), 그리고 Framer Motion을 활용한 부드러운 인터랙션이 특징입니다.

- **Tech Stack:** React, TypeScript, Tailwind CSS, Framer Motion, Vite
- **Design Focus:** 깔끔한 아이디어, 고객과의 활발한 교류, 빠른 연락

## 🌟 Key Highlights

- **Interactive Business Card:** 마우스의 움직임에 따라 반응하는 3D 명함 인터랙션 구현
- **Premium Typography:** 국내 탑티어 웹 서비스에서 사용되는 `Pretendard`와 트렌디한 영문 폰트 `Outfit` 조합
- **Smooth Animations:** 사용자 경험(UX)을 극대화하는 스크롤 진행바 및 페이드 인/아웃 효과
- **Projects & Communities:** `Design Pick`, `나랏말싸미(훈민정음)`, `HANN봇` 등 제가 리드하고 있는 다채로운 프로젝트들과 총 2,000명 이상의 유저와 교감해 온 디스코드 커뮤니티 운영 기록 (`로블갤러리`, `Limited™`, `로폴더`)

## 💻 Local Development

이 프로젝트를 로컬에서 직접 실행해보고 싶으시다면 다음 명령어를 사용해 주세요:

```bash
# 레포지토리 클론
git clone https://github.com/haroseo/Seoharo.git

# 프로젝트 폴더 이동
cd Seoharo

# 의존성 패키지 설치
npm install

# 개발 서버 실행
npm run dev
```
